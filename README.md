# AURA FLORA

A PHP, CSS, and JavaScript floral-commerce prototype with an AI event preview simulator.

## Run with XAMPP

Place the project in `htdocs` and open the project URL through Apache.

## Deploy to Vercel

1. Import the GitHub repository into Vercel.
2. Leave the Framework Preset set to **Other**.
3. Leave Build Command and Output Directory empty.
4. Deploy. The included `vercel.json` uses the community PHP runtime and routes requests to `api/index.php`.

The browser-facing demo data is stored in `localStorage`; no database or environment variables are required.
